PROJECT TITLE: Capstone game
PURPOSE OF PROJECT: to synthesize multiple concepts into a cool game
VERSION or DATE: 5.10.16
HOW TO START THIS PROJECT: run the main method of the GameViewer class
AUTHORS: Nic Gardiner (with keyListener help from Nic Guerrero)
USER INSTRUCTIONS: use the left and right arrow keys to move the aiming device, shoot with up or spacebar
OVERVIEW:   This project creates a frame on which the game can be played.  The game
            has a cluster of multicolored bubbles which pop in groups of the same color
            (once hit from the bubble the user fires).
EXPLANATION: Synthesizes many concepts, such as the use of arrays, object oriented design,
             recursion (kind of), and more to create this game.
